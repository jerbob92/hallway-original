1. Connect your GitHub account if you haven't yet (under services in upper right corner)
  2. Fork the Skeleton Viewer
  3. Add a js/config.js file to include your API Token
  4. Edit and test your viewer!
    • Loading your index.html locally should work :)
    • Change your .app file to customize your viewer name
    • Push your changes back to github, and check them out from inside your locker
  5. Use the API Explorer to discover lots of your data to work with
